# Domain Packs

## Purpose
Provide reusable domain-specific starter assets that can be attached with framework standards during AI prompting.

---

# Included Packs

- healthcare
- insurance
- finance

Each pack contains:

- domain glossary
- domain business rules
- sample requirement

---

# Usage

When generating artifacts:

1. Attach core framework standards.
2. Attach one domain pack (or combine as needed).
3. Run bootstrap prompt.
4. Run generation and review prompts.
