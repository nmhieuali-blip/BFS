# Healthcare Domain Glossary

| Term | Meaning |
|---|---|
| EMR/EHR | Electronic medical record |
| Encounter | A patient clinical visit |
| Triage | Priority assessment at intake |
| ICD-10 | Diagnostic coding standard |
| Prior Authorization | Insurer pre-approval for treatment |
| Referral | Request from primary provider to specialist |
| Clinical Note | Provider documentation for encounter |
| Disposition | Patient outcome after visit |
