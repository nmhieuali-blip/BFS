# Sample Requirement - Insurance

## Requirement ID
REQ-INS-001

## Feature
Claim Resubmission After Denial

## Problem
Denied claims are resubmitted without clear correction workflow, causing repeated denials.

## Goal
Standardize correction, validation, and controlled resubmission with full traceability.

## Constraints
- Duplicate prevention required
- Denial code mapping required
- Retry and timeout handling required
