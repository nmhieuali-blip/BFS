# Insurance Domain Glossary

| Term | Meaning |
|---|---|
| Claim | Request for reimbursement/payment |
| Policy | Contract defining insured coverage |
| Deductible | Amount patient pays before insurer coverage |
| Copay | Fixed amount paid per service |
| Pre-authorization | Approval required before service |
| Adjudication | Insurer decision process for claim |
| EOB | Explanation of Benefits |
| Denial Code | Standardized rejection reason |
