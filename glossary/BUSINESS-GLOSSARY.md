# BUSINESS GLOSSARY

| Term | Meaning |
|---|---|
| HIS | Hospital Information System |
| Encounter | Medical visit |
| Claim | Insurance payment request |
| Admission | Patient check-in |
| Discharge | Patient checkout |
| Appointment | Scheduled patient visit |
| Invoice | Billing document |
| RBAC | Role-Based Access Control |
| Permission | Granular action authorization |
| Access Token | JWT used for API access |
| Refresh Token | Token used to renew access token |
| Gateway | Central API routing layer |
| BPMN | Business Process Model and Notation |
| Sequence Diagram | Interaction flow diagram |
| State Machine | State transition model |
| Retry | Re-execution after failure |
| Timeout | Maximum waiting duration |
| Compensation | Rollback action |
| Event | Async communication message |
| Queue | Message broker channel |
