# USER STORY STANDARD

## Purpose

Standardize:
- User story format
- Naming convention
- Priority
- Dependencies
- Business rule linkage

---

# Standard Format

## Story ID

Format:

US-{MODULE}-{NUMBER}

Example:

US-AUTH-001

---

# Story Title

Format:

[MODULE] - [ACTION] - [TARGET]

Example:

[AUTH] Login with username/password

---

# Story Template

As a [actor]

I want [action]

So that [business value]

---
